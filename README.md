--------------------------------------------             Music Player          ------------------------------------------------------

Play music from a curated list of public domain tracks.

Features :

  Shows the song's name, the album it's from, the artist and it's album image.

  Pause, skip to next song, go back to previous song. 

  Click on the song's progress bar to skip to a certain section.

  Turn up and down the volume, until it's completely muted.


Project evolution ideas :

- Implement a playlist system
- Give the user the power to upload and play it's own music file
